# 01 - Java Basics

## 🧠 Why Java?
- Platform independent
- Object-oriented
- GSoC supports it heavily (Android SDKs)

## 🔑 Keywords
- `class`, `public`, `static`, `void`, `main`, `System.out.println()`

## 🔍 Sample Code
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Ujwal!");
    }
}
